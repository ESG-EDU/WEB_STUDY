### JOIN 

| JOIN 종류 | 설명 | 반환되는 결과 | 특징 |
| :---: | :---: | :---: | :---: |
| **INNER JOIN** | 두 테이블에서 **조건이 일치하는 행만** 가져옴 | 교집합 | 가장 많이 사용됨. 일치하지 않는 데이터는 모두 제외됨 |
| **LEFT JOIN (LEFT OUTER JOIN)** | 왼쪽 테이블의 모든 행 + 오른쪽과 조건 일치하는 행 | 왼쪽 기준 전체 데이터 | 오른쪽에 일치 없음 → NULL로 채움 |
| **RIGHT JOIN (RIGHT OUTER JOIN)** | 오른쪽 테이블의 모든 행 + 왼쪽과 조건 일치하는 행 | 오른쪽 기준 전체 데이터 | MySQL/MariaDB에서는 사용 가능하지만 LEFT JOIN보다 덜 사용 |
| **CROSS JOIN** | 테이블 간 모든 조합을 반환 (Cartesian Product) | A × B (곱집합) | 조합 수가 매우 커질 수 있음. ON 조건을 사용하지 않음 |
