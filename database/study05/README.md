### [그룹함수] 집계함수

| 함수 | 설명 | 
| :---: | :---: |
| count(*) | 로우 개수 - null 포함 |
| count(컬럼명) | 로우 개수 - null 제외 |
| sum(컬럼명) | 합 |
| avg(컬럼명) | 평균 |
| max(컬럼명) | 최댓값 |
| min(컬럼명) | 최솟값 |

> 문법
```sql
select 컬럼명, 그룹함수(컬럼명)
from 테이블명
where 조건
group by 컬럼명
having 조건
```
